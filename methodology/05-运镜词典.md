# 05 · 运镜词典

> 速查用。模型对运镜术语理解很强，**直接用标准术语**即可，例如"中景"、"特写"、"全景"、"缓慢推镜"、"平稳横移"、"固定镜头"。

## 镜头大小

| 英文术语 | 中文 | 适用场景 |
|---|---|---|
| Extreme Wide Shot (EWS) | 极广角 | 地理、孤立感、宏大场景 |
| Wide Shot (WS) | 广角 | 环境交代、地理位置 |
| Medium Shot (MS) | 中景 | 关系、手势 |
| Medium Close-Up (MCU) | 中特写 | 意图、手部、工具 |
| Close-Up (CU) | 特写 | 情感、细节 |
| Extreme Close-Up (ECU) | 极特写 | 纹理、眼睛、微表情 |

## 镜头运动

| 英文术语 | 中文 | 效果 |
|---|---|---|
| Dolly in / push in | 推镜 | 强调、聚焦 |
| Dolly out / pull back | 拉镜 | 揭示、疏离 |
| Pan left / right | 横摇 | 跟随、展示空间 |
| Tilt up / down | 纵摇 | 揭示高度 |
| Crane up / down | 升降 | 宏观视角 |
| Tracking shot | 跟踪镜头 | 跟随主体运动 |
| Orbit / arc | 环绕 | 产品展示、角色介绍 |
| Handheld | 手持 | 纪实感、紧张感 |
| Gimbal | 稳定器 | 流畅运动 |
| Steadicam | 斯坦尼康 | 长镜头跟随 |
| Rack focus | 焦点转换 | 引导注意力 |
| Slow push-in | 缓慢推进 | 建立悬念 |
| Aerial / drone | 航拍/无人机 | 宏观视角 |

> ⚠️ 一个镜头里**尽量只指定 1 种运镜方式**，不要同时要求推拉摇移，会增加画面的不稳定性。

## 灯光

| 英文术语 | 中文 | 情绪效果 |
|---|---|---|
| Golden hour | 黄金时刻 | 温暖、怀旧、旅行感 |
| Blue hour | 蓝调时刻 | 冷峻、科幻、孤独 |
| Overcast | 阴天漫射光 | 纪实、真实、均匀 |
| Hard key light | 硬光 | 戏剧性、黑色电影 |
| Soft window light | 柔和窗光 | 平静、亲密、真实 |
| Neon spill | 霓虹溢光 | 赛博朋克、夜生活 |
| Volumetric lighting | 体积光 | 史诗感、神圣感 |
| Backlight / rim light | 背光 / 轮廓光 | 剪影、轮廓强调 |
| Practical light | 实景灯光 | 自然、真实 |
| Studio lighting | 影棚灯光 | 商业、产品 |
| Butterfly lighting | 蝴蝶光 | 美容、人像、高端 |
| Tungsten | 钨丝灯（暖黄） | 室内、复古 |
| Fluorescent | 荧光（冷白） | 办公、地铁、医院 |

## 风格

| 风格 | 英文关键词 |
|---|---|
| 电影感 | cinematic, film grain, anamorphic, 35mm |
| 纪录片 | documentary, handheld, naturalistic, vérité |
| 商业广告 | commercial quality, clean aesthetic, product hero |
| 赛博朋克 | cyberpunk, neon-lit, holographic, dystopian |
| 复古胶片 | vintage, 8mm, VHS, retro grain, color shift |
| 动漫风格 | anime aesthetic, cel-shaded, Ghibli-inspired |
| 水墨风格 | ink wash, Chinese painting, calligraphy style |
| 超现实 | surreal, dreamlike, Escher-inspired, impossible geometry |
| 黑色电影 | film noir, high contrast B&W, deep shadows |
| 韦斯安德森 | Wes Anderson, symmetrical composition, pastel palette |
