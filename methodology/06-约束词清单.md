# 06 · 约束词清单

> 约束词十分重要，可有效规避画面瑕疵、畸形崩坏及不合理元素，**约束生成边界与稳定性**。

## 必加的 3 个基础约束

每条提示词都建议加（除非你明确需要这些元素）：

```
不要生成水印
不要生成Logo
保持无字幕 / 避免生成任何文字或字幕
```

## 按问题分类的约束词

### 画面稳定性

```
人物面部稳定不变形
身体比例稳定不变形
动作连贯自然，不僵硬
无穿模、无卡顿、无闪烁
```

### 风格一致性

```
全程保持 [具体风格] 风格
风格不漂移
色调一致
```

### 主体一致性（防止双胞胎）

```
视频全程禁止出现外形、着装、配饰完全一致的人物
禁止生成同款分身、双胞胎效果
同一画面中仅保留单个对应人物，不出现人物重复复刻
```

### 镜头稳定性

```
镜头平稳，无剧烈晃动
固定机位
避免突然变焦
```

### 音频干净度

```
音频自然，无截断噪音
片尾音频自然淡出
```

## 反向约束（明确告诉模型不要做什么）

英文场景：

```
avoid static shots
avoid camera shake
no text overlays
no watermarks
avoid blurry faces
no extra limbs
no distorted hands
```

## 矛盾约束清单（不要叠加）

这些组合**会相互抵消**，写了等于没写：

| ❌ 矛盾组合 | 原因 |
|---|---|
| `8mm 胶片感` + `4K 超清` | 胶片感本身就是模糊+颗粒，和超清冲突 |
| `film grain` + `ultra sharp` | 同上 |
| `电影感` + `纪实手持` 都强写 | 风格相互拉扯 |
| 同时指定多个艺术家风格（>2 个） | 模型会随机取一个 |
| `slow motion` + `high speed action` | 速度矛盾 |

## 完整模板

```
[主体 + 动作 + 场景 + 光影 + 镜头 + 风格 + 画质]
…正向描述…

约束：
- 人物面部稳定不变形
- 动作连贯自然，不僵硬
- 全程保持 [风格] 风格，风格不漂移
- 不要生成水印、Logo、字幕
```
